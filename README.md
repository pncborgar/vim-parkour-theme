## Installation With vim-plug 

Add this line to your vim/neovim config file:

`Plug 'pncborgar/vim-parkour-theme'`

and than in vim, run:

`:PlugInstall`

and then to set the parkour theme as default vim theme, add this to your vim config:

`colorscheme parkour`
